# 🧪 Sandbox — Experimental Channel Tools

> **Warning:** Everything in here is unvetted, experimental, and likely broken. These are nightly R&D prototypes — not production-ready tools.

This directory is generated by a **nightly automated research & build cron job** that:

1. **Researches** — scans current channel management trends, partner program innovations, AI/automation patterns, and competitive intelligence
2. **Ideates** — designs new tools that could solve channel problems (deal registration, partner onboarding, MDF tracking, lead routing, ROI forecasting, etc.)
3. **Builds** — creates working prototypes as standalone scripts, MCP servers, web apps, or CLI tools
4. **Delivers** — pushes to this directory every night so you wake up to fresh ideas

## Guidelines

- **No constraints** — not limited by Chanimal docs or existing tools. Think bigger.
- **Anything goes** — MCP servers, web micro-apps, CLI tools, automation scripts, AI agents
- **Clean commits** — each nightly session gets its own commit with a descriptive message
- **Zero polish required** — rough prototypes > nothing. The goal is exploration.
